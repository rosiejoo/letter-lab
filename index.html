IGAW LetterLab 가이드

처음이라면 여기서 시작하세요

1Gemini API 키 설정

뉴스레터 생성에 Gemini AI를 사용해요. 먼저 API 키가 필요합니다.

키 발급aistudio.google.com/apikey에서 Create API Key 클릭
등록우측 상단 ⚙ 설정에 붙여넣고 저장하면 준비 끝
2뉴스레터 생성

뉴스레터로 만들고 싶은 콘텐츠 URL만 넣으면 AI가 알아서 작성해요.

플랫폼해당 콘텐츠의 솔루션을 선택하세요
스타일소제목형(기본)과 줄글형 중 선택
분량"소제목 3개", "5줄" 등 원하는 만큼 조절 가능
트래킹 링크입력 시 해당 섹션의 이미지·원문 보기 링크로 자동 연결
링크 추가URL 여러 개로 멀티 섹션 뉴스레터도 가능
발행일생성 버튼 옆 날짜로 지정

💡 수신거부·블로그 아티클 바로가기 링크에는 트래킹 링크가 자동으로 적용됩니다.

3편집 모드

Edit 버튼을 누르면 뉴스레터를 자유롭게 수정할 수 있어요. 우측에 도구 패널이 열리고, 본문의 요소를 클릭하면 바로 편집됩니다.

텍스트클릭하면 바로 수정
이미지클릭해서 교체, 우하단 드래그로 크기 조절
박스클릭하면 우측 패널 상단에 설정이 나타나요
구분선클릭해서 색상·두께·스타일 변경
섹션 순서여러 URL일 때 드래그로 순서 변경

📝 기본 글씨체 Noto Sans KR · 본문 16px · 소제목 18px Bold

4부분 내용 수정

전체가 아니라 특정 문장만 고치고 싶을 때 유용해요. 고치고 싶은 부분을 드래그로 선택하면 수정 요청 버튼이 뜹니다.

"더 짧게", "수치 강조", "톤 부드럽게" 같이 입력하면 AI가 그 부분만 다시 써줘요.

5이미지

이미지는 AI가 자동으로 넣어주지 않아요. 편집 모드에서 직접 추가하거나 교체해야 합니다.

이미지 추가우측 도구 패널 → 삽입 → 이미지 클릭
이미지 교체본문의 이미지 클릭 → 모달에서 새 이미지 설정
크기 조절이미지 우하단 모서리 드래그
클릭 링크이미지 클릭 시 이동할 URL 설정 가능

이미지 소스별 사용 가능 여부

✓파일 업로드 — 자동으로 URL 변환됨 (imgbb)
✓Wix 블로그 — 이미지 우클릭 → 이미지 주소 복사
✓외부 이미지 URL — https://로 시작하는 직접 링크
✗Google Drive — 공유 링크는 이메일에서 안 보임
✗복사-붙여넣기 — base64 형식은 이메일에서 차단됨

💡 파일 업로드 후 '업로드 완료' 표시를 확인하고 반드시 "적용" 버튼을 눌러주세요!

SNS 공유 텍스트 작성 도구

1텍스트 생성

SNS에 공유할 콘텐츠 URL을 입력하고 생성 버튼을 누르면, 페이스북·링크드인·인스타그램용 텍스트가 3개씩 생성됩니다.

각 후보는 서로 다른 각도(수치 강조, 스토리텔링, 질문형 등)로 작성돼요.

2후보 카드

3개 후보가 나란히 표시돼요. 각 카드에서 바로 사용할 수 있습니다.

📋 복사해당 후보 텍스트를 클립보드에 복사
👁 미리보기실제 SNS 포스트처럼 미리 확인
✏️ 수정모달에서 직접 편집, 톤/스타일 변경, 재생성, 원본 대조, 저장
3수정 모달

카드에서 "수정"을 누르면 열리는 편집 화면이에요.

✏️ 직접 편집텍스트를 클릭해서 바로 수정
🎭 톤 변경캐주얼 / 전문적 / 후킹
🎨 스타일수치 강조 / 스토리텔링 / 질문형 / 문제의식 / 반전 포인트
🔄 재생성톤+스타일 선택 후 재생성 버튼
📑 원본 대조SNS 텍스트와 원문 핵심 문장을 좌우로 비교
4조합

여러 후보에서 마음에 드는 문장만 골라서 하나의 텍스트로 합칠 수 있어요.

문장 선택후보 카드에서 원하는 문장을 드래그로 선택
조합함 추가선택하면 뜨는 "📌 조합함에 추가" 버튼 클릭
조합 버튼툴바의 "조합" 버튼으로 조합함 패널 열기
순서 변경조합함에서 드래그로 문장 순서 조정
AI 다듬기문맥이 자연스럽게 이어지도록 AI가 정리
복사/저장완성된 조합 텍스트를 복사하거나 저장

💡 1번 후보의 도입부 + 2번 후보의 데이터 + 3번 후보의 마무리 이런 식으로 조합하면 최적의 텍스트를 만들 수 있어요!
