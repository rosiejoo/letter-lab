IGAW LetterLab 가이드

처음이라면 여기서 시작하세요

1️⃣ Gemini API 키 설정
뉴스레터 생성에 Gemini AI를 사용해요. 먼저 API 키가 필요합니다.

키 발급 aistudio.google.com/apikey에서 Create API Key 클릭
등록 우측 상단 ⚙ 설정에 붙여넣고 저장하면 준비 끝
2️⃣ 뉴스레터 생성
뉴스레터로 만들고 싶은 콘텐츠 URL만 넣으면 AI가 알아서 작성해요.

플랫폼해당 콘텐츠의 솔루션을 선택하세요 스타일소제목형(기본)과 줄글형 중 선택 분량"소제목 3개", "5줄" 등 원하는 만큼 조절 가능 트래킹 링크입력 시 해당 섹션의 이미지·원문 보기 링크로 자동 연결 링크 추가URL 여러 개로 멀티 섹션 뉴스레터도 가능 발행일생성 버튼 옆 날짜로 지정
수신거부·블로그 아티클 바로가기 링크에는 트래킹 링크가 자동으로 적용됩니다.
3️⃣ 편집 모드
Edit 버튼을 누르면 뉴스레터를 자유롭게 수정할 수 있어요.
우측에 도구 패널이 열리고, 본문의 요소를 클릭하면 바로 편집됩니다.

텍스트클릭하면 바로 수정 이미지클릭해서 교체, 우하단 드래그로 크기 조절 박스클릭하면 우측 패널 상단에 설정이 나타나요 구분선클릭해서 색상·두께·스타일 변경 섹션 순서여러 URL일 때 드래그로 순서 변경
기본 글씨체 Noto Sans KR · 본문 16px · 소제목 18px Bold
4️⃣ 부분 내용 수정
전체가 아니라 특정 문장만 고치고 싶을 때 유용해요.
고치고 싶은 부분을 드래그로 선택하면 수정 요청 버튼이 뜹니다.
"더 짧게", "수치 강조", "톤 부드럽게" 같이 입력하면 AI가 그 부분만 다시 써줘요.
5️⃣ 이미지
✓ 파일 업로드 → 자동 URL 변환
💡 '업로드 완료' 표시 확인 후 반드시 적용 클릭
✓ Wix에서 이미지 우클릭 → 주소 복사
✗ Google Drive 링크 — 이메일에서 안 보임
✗ 복사-붙여넣기(base64) — 이메일에서 차단
6️⃣ 내보내기 & 저장
Copy HTML복사해서 Stibee에 바로 붙여넣기 임시저장아직 보내기 전이라면 저장해두고 나중에 이어서 편집 미리보기실제 이메일처럼 600px 기준으로 미리 확인
⚠️ 알아두면 좋은 것들
·원본 대조는 방금 만든 뉴스레터만 가능해요 ·Gemini API 무료 제한이 있어서, 많이 돌리면 결과가 이상해질 수 있어요 ·이미지는 AI가 넣어주지 않아요. 직접 교체하거나 추가해야 해요 ·임시저장·히스토리는 브라우저에 저장돼요. 다른 기기에서는 안 보여요 ·API 키도 브라우저에 남아요. 공용 PC라면 사용 후 삭제해주세요 ·Gmail은 HTML 102KB 넘으면 잘려요. 이미지는 URL로, 공백은 줄여주세요
