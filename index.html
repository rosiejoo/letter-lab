뉴스레터 생성

만들고 싶은 콘텐츠 URL을 넣으면 AI가 뉴스레터를 자동으로 작성해요.

플랫폼해당 콘텐츠의 솔루션 태그를 선택하세요
스타일소제목형(기본) · 줄글형 중 선택
분량소제목형: "소제목 3개", "소제목 5개" / 줄글형: "❶❷❸ 3개", "500자" / 공통: "5줄", "3줄"
발행일URL 옆 날짜 필드에서 발행 예정일 지정 — 뉴스레터 상단에 자동 표시
트래킹 링크이미지 클릭·원문 보기 링크에 자동 연결됨
멀티 URL링크 여러 개로 멀티 섹션 뉴스레터 생성 가능

수신거부·아티클 바로가기 링크에는 트래킹 링크가 자동 적용됩니다.

🔗 1단계 — URL 입력 & 텍스트 생성

홍보하고 싶은 콘텐츠의 URL을 입력하고 생성 버튼을 누르면, 페이스북·링크드인·인스타그램 각 플랫폼에 맞는 텍스트가 3개씩 만들어져요.

URL은 최대 5개까지 한번에 넣을 수 있어요. 여러 콘텐츠를 조합한 포스트도 가능합니다.

🏷️ 2단계 — 후보 카드 이해하기

각 후보 카드 상단에는 라벨이 붙어 있어요. AI가 콘텐츠를 분석해서 가장 적합한 3가지 작성 방향을 자동으로 골라줍니다.

📊 수치 강조데이터·통계로 신뢰도 UP
📖 스토리텔링서사로 공감·감성 유도
⚡ 문제의식독자의 고민에서 시작
❓ 질문형궁금증으로 클릭 유도
🔄 반전 포인트예상 밖 결과로 훅 걸기
🛠 실용 전략바로 쓸 수 있는 인사이트

💡 라벨은 AI가 자동 선택해요. "수정" 버튼에서 톤·스타일을 바꿔 재생성할 수도 있어요.

✏️ 3단계 — 수정 & 재생성

카드의 수정 버튼을 누르면 편집 모달이 열려요. 여기서 할 수 있는 것들:

직접 수정텍스트를 직접 클릭해서 자유롭게 편집
재생성톤(캐주얼/전문적/후킹) + 스타일(수치 강조/스토리텔링/질문형/문제의식/반전) 선택 후 재생성
수정 완료편집이 끝나면 ✓ 수정 완료 버튼으로 카드에 반영
🔎 4단계 — 미리보기 & 복사

카드의 🔎 미리보기 버튼을 누르면 실제 SNS에 올렸을 때의 모습을 확인할 수 있어요.

마음에 드는 후보가 있으면 복사 버튼으로 바로 클립보드에 복사하세요.

🧩 5단계 — 조합 (내 입맛대로 섞기)

3개 후보에서 좋은 문장만 골라 하나로 합칠 수 있어요.

예를 들어 1번의 도입부 + 2번의 데이터 + 3번의 마무리 이런 식으로요!

① 문장 선택카드에서 원하는 문장을 드래그로 선택
② 조합함 추가나타나는 "📌 조합함에 추가" 버튼 클릭
③ 조합 열기툴바의 🧩 조합 버튼 → 순서를 드래그로 조정
④ AI 다듬기문맥이 자연스럽게 이어지도록 AI가 정리해줘요
⑤ 그대로 복사다듬기 없이 선택한 문장 그대로 복사도 가능
🔖 6단계 — 저장 & 관리
카드 저장카드의 북마크 아이콘 클릭 → 저장 목록에 보관
저장 취소채워진 북마크를 다시 클릭하면 저장 취소
저장 목록클릭하면 전체 텍스트 편집·재생성·복사 가능
히스토리생성할 때마다 자동 저장, 이전 결과 언제든 불러오기
💡 꿀팁
  • 저장·히스토리는 브라우저에 저장돼요. 다른 기기에서는 안 보여요
  • Gemini API 무료 한도 초과 시 잠시 기다렸다 다시 시도해주세요
